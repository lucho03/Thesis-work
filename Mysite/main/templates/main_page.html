About this project

This project aims to get as close as possible to the real products available on the market today. The main communication is done through two channels: within the system and through emails. There are two main roles in the system: user and agent. The user can send and edit tickets, receive replies and add additional information to the ticket and reply. The agent has the ability to reply, close and delete tickets. Replying can be done inside the system or by sending an email. Also by email come the tickets and additional information to them. The project is course work in grade 12.
